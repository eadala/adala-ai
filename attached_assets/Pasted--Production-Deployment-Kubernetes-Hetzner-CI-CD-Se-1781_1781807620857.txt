☸️ رابعاً: Kubernetes Deployments

🧠 AI Core Deployment

apiVersion: apps/v1
kind: Deployment
metadata:
  name: ai-core
spec:
  replicas: 3
  selector:
    matchLabels:
      app: ai-core
  template:
    metadata:
      labels:
        app: ai-core
    spec:
      containers:
        - name: ai-core
          image: registry/ai-core:latest
          ports:
            - containerPort: 4000
          env:
            - name: SHUAIB_URL
              value: "https://shuaib.internal"

🔗 خامساً: API Gateway (النقطة المركزية)

📌 مسؤولياته:

* Authentication
* Rate limiting
* Routing
* Tenant isolation
* Audit logging

⸻

مثال NestJS Guard (صلاحيات قضائية)

@Injectable()
export class LegalGuard implements CanActivate {
  canActivate(context: ExecutionContext): boolean {
    const request = context.switchToHttp().getRequest();
    if (!request.user.officeId) {
      throw new ForbiddenException('غير مصرح: بدون مكتب قضائي');
    }
    return true;
  }
}

🧾 سابعاً: نظام الحوكمة القضائية (Governance Layer)

📌 Audit Log إلزامي

export function auditLog(action: string, payload: any) {
  return prisma.audit.create({
    data: {
      action,
      payload,
      timestamp: new Date(),
    },
  });
}

🔐 ثامناً: الأمان (Security Hardening)

1. Network Isolation

* Private subnet بين الخدمات
* DB غير مكشوف للعامة

2. JWT + RBAC

* أدوار قضائية صارمة

3. Rate Limiting

* منع استغلال AI

4. Encryption

* AES-256 للوثائق
