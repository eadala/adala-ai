تمام — هذه هي مرحلة بناء لوحة التحكم الفعلية (Production Admin Control Panel) لمنصة “عدالة” بشكل احترافي جاهز للتطوير والنشر، مع ربط كامل بنظام الاشتراكات + المكاتب + الذكاء الاصطناعي + نظام الشعيب.

سأبنيها لك كـ Enterprise SaaS Admin Panel Architecture + React Code Skeleton فعلي.

⸻

🧭 لوحة التحكم الإدارية — عدالة (Admin Control Panel)

🏛️ الهدف

لوحة مركزية لإدارة:

* المكاتب (Tenants)
* الاشتراكات

export async function getTenants() {
  return fetch("/api/admin/tenants").then(r => r.json());

export async function suspendTenant(id: string) {
  return fetch(`/api/admin/tenants/${id}/suspend`, {
    method: "POST"
  });
}

export function requireRole(role: string) {
  return (req, res, next) => {
    if (req.user.role !== role) {
      return res.status(403).send("غير مصرح");
    }
    next();
  };
}

⚖️ القاعدة النظامية النهائية

📌 المادة (1):

تُدار جميع المكاتب القانونية عبر نظام تعدد المستأجرين (Multi-Tenant) مع عزل كامل للبيانات.

📌 المادة (2):

تخضع جميع عمليات الذكاء الاصطناعي للمراقبة والتسجيل لأغراض الفوترة والحوكمة.

📌 المادة (3):

لا يجوز إيقاف أو تعديل أي بيانات قضائية في نظام الشعيب عبر لوحة التحكم.

⸻
