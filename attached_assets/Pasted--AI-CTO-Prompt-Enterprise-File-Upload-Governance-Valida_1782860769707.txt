# AI CTO Prompt — Enterprise File Upload Governance & Validation

## المهمة

بعد إصلاح مشكلة Upload Dialog في مركز المستندات، انتقل إلى مرحلة حوكمة نظام رفع الملفات بالكامل (Enterprise File Upload Governance).

الهدف ليس إصلاح نافذة واحدة، وإنما ضمان أن جميع عمليات رفع الملفات في منصة عدالة تعمل بشكل صحيح ومتسق على جميع الوحدات وجميع الأجهزة.

# المرحلة الثامنة: Error Handling

أي خطأ أثناء الرفع يجب ألا يؤدي إلى:

- سقوط الصفحة
- ظهور Root AppErrorBoundary
- Crash للتطبيق

بدلاً من ذلك:

يعرض رسالة واضحة داخل نافذة الرفع مع:

سبب الخطأ

زر إعادة المحاولة

زر الإلغاء

ويستمر التطبيق بالعمل بشكل طبيعي.

✓ جميع اختبارات Regression ناجحة.

✓ تم إصدار تقرير شامل يوضح عدد نقاط رفع الملفات التي تم فحصها، والنتائج، وأي ملاحظات أو تحسينات مقترحة.

الهدف النهائي هو اعتماد نظام رفع ملفات موحد، آمن، وقابل للتوسع على مستوى منصة عدالة بالكامل، بحيث تستخدم جميع الوحدات نفس المعايير والمكونات الأساسية دون تكرار أو اختلاف في السلوك.
