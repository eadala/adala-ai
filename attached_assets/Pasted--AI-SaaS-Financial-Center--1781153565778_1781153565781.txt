إذا كانت **عدالة AI** منصة SaaS متعددة المكاتب، فالطريقة الاحترافية هي بناء **Financial Center** وربطه بـ **Stripe Connect** بحيث:

* العميل يدفع للمكتب.
* عدالة AI تقتطع 10%.
* Stripe يحول صافي المبلغ للمكتب.
* يحتفظ النظام بسجل كامل للحوالات والمدفوعات.

## `shared/schema/officeBankAccounts.ts`

```ts
import {
  pgTable,
  uuid,
  varchar,
  timestamp
} from "drizzle-orm/pg-core";

export const officeBankAccounts = pgTable(
  "office_bank_accounts",
  {

    id: uuid("id")
      .defaultRandom()
      .primaryKey(),

    officeId: uuid("office_id")
      .notNull(),

    beneficiaryName: varchar(
      "beneficiary_name",
      {
        length: 255
      }
    ).notNull(),

    bankName: varchar(
      "bank_name",
      {
        length: 255
      }
    ),

    iban: varchar(
      "iban",
      {
        length: 50
      }
    ),

    accountNumber: varchar(
      "account_number",
      {
        length: 100
      }
    ),

    swiftCode: varchar(
      "swift_code",
      {
        length: 50
      }
    ),

    currency: varchar(
      "currency",
      {
        length: 10
      }
    ).default("SAR"),

    createdAt: timestamp(
      "created_at"
    ).defaultNow()

  }
);
```

# 4. إنشاء حساب Stripe Connect للمكتب

## `server/services/stripeConnectService.ts`

```ts
import { stripe } from "../lib/stripe";

export async function createConnectedAccount(
  email: string
) {

  const account =
    await stripe.accounts.create({

      type: "express",

      email,

      capabilities: {

        transfers: {
          requested: true
        },

        card_payments: {
          requested: true
        }

      }

    });

  return account;

}
```

```ts
export async function createOnboardingLink(
  accountId: string
) {

  return await stripe.accountLinks.create({

    account: accountId,

    refresh_url:
      "https://adala-ai.com/reconnect",

    return_url:
      "https://adala-ai.com/dashboard",

    type: "account_onboarding"

  });

}
```

# 6. إنشاء عملية دفع

## خصم 10%

```ts
export async function createPaymentIntent(

amount: number,

connectedAccountId: string

) {

  return await stripe.paymentIntents.create({

    amount,

    currency: "sar",

    application_fee_amount:

      Math.floor(amount * 0.10),

    transfer_data: {

      destination:
        connectedAccountId

    }

  });

}
```
