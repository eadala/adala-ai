ADALA AI — UI/UX Improvement & Mobile Optimization (Non-Breaking Update)

قم بإجراء تحسينات شاملة على واجهة منصة عدالة AI الحالية دون تغيير البنية الأساسية أو كسر أي وظائف موجودة.

⸻

🎯 الهدف

تحسين تجربة المستخدم (UX) وتصميم الواجهة (UI) مع التركيز على:

* الجوال (Mobile Experience)
* الاتساق البصري
* الأداء
* إصلاح مشاكل التفاعل (Interaction Bugs)

بدون تعديل منطق النظام الأساسي أو قواعد البيانات.

⸻

📱 1. Mobile UI Fixes (أولوية قصوى)

المطلوب:

* ضمان أن جميع الصفحات Responsive بالكامل
* منع أي Overflow أفقي (Horizontal Scroll)
* تحسين عرض:
    * القضايا
    * العملاء
    * الفواتير
    * التذاكر
    * الملفات

تحويل تلقائي:

* Tables → Cards على الجوال
* Sidebars → Drawer
* Modals → Bottom Sheet عند الحاجة

⸻

🖼️ 2. Image Viewer Fix (Critical Bug Fix)

المشكلة الحالية:

* الصور تفتح Full Screen لكن:
    * لا يمكن الخروج منها بسهولة
    * أو يتجمد التمرير بعد الإغلاق
    * أو تختفي عناصر التحكم

المطلوب:

* Full Screen Image Viewer احترافي
* دعم:
    * زر إغلاق واضح (X)
    * Swipe Down للإغلاق
    * زر الرجوع في الجوال
    * Pinch Zoom
    * Swipe بين الصور

إصلاح:

* منع body scroll lock بعد الإغلاق
* تنظيف state عند close
* إعادة تفعيل التفاعل بعد الخروج

⸻

🎨 3. UI Consistency

* توحيد:
    * الخطوط
    * المسافات
    * حجم الأزرار
    * الكروت
* تحسين readability على الجوال
* تقليل ازدحام العناصر داخل الصفحات

⸻

📊 4. Data Display Improvements

* تحسين عرض البيانات داخل:
    * الجداول
    * القوائم
    * البطاقات

المطلوب:

* إبراز البيانات المهمة
* تقليل المعلومات الثانوية
* تحسين قابلية القراءة السريعة

⸻

🧭 5. Navigation (Mobile)

* تحويل Sidebar إلى Drawer
* تحسين التنقل بين الصفحات
* إضافة دعم Back navigation واضح
* إغلاق القوائم تلقائياً بعد الاختيار

⸻

💳 6. Finance & Tickets UI

تحسين عرض:

* الفواتير
* المدفوعات
* التذاكر

المطلوب:

* تصميم واضح للجوال
* عرض الحالة (Paid / Pending / Open) بشكل بصري
* تبسيط التفاصيل داخل الكروت

⸻

⚡ 7. Performance Optimization

* تحسين تحميل الصور (Lazy Loading)
* تقليل re-render غير الضروري
* تحسين سرعة فتح الصفحات
* تقليل استهلاك الذاكرة على الجوال

⸻

🔐 8. UX Stability Fixes

* منع تجمّد الصفحة بعد Modals أو Image Viewer
* ضمان إمكانية الرجوع في جميع الحالات
* منع UI Lock نهائيًا
* إصلاح أي تعارض بين overlays

⸻

🏁 SUCCESS CRITERIA

يجب أن تصبح واجهة منصة عدالة AI:

* سلسة بالكامل على الجوال
* بدون أخطاء UI أو تجمّد
* متناسقة بصريًا
* سريعة الاستجابة
* سهلة الاستخدام للمحامين والعملاء
* دون أي تغييرات على النظام الخلفي

نفّذ التحسينات فقط دون تعديل البنية الأساسية أو قواعد البيانات أو API.
