# جدول الخدمات

## `shared/schema.ts`

```ts
export const services = pgTable("services", {

  id: uuid("id").defaultRandom().primaryKey(),

  officeId: uuid("office_id").notNull(),

  name: varchar("name", { length: 255 }).notNull(),

  slug: varchar("slug", { length: 255 }).notNull(),

  shortDescription: text("short_description"),

  description: text("description"),

  price: decimal("price"),

  durationDays: integer("duration_days"),

  imageUrl: text("image_url"),

  isFeatured: boolean("is_featured").default(false),

  isActive: boolean("is_active").default(true),

});
```

# صفحة الخدمات

## `pages/ServicesPage.tsx`

```tsx
import { useQuery } from "@tanstack/react-query";
import { ServiceCard } from "@/components/services/ServiceCard";

export default function ServicesPage() {

  const { data = [] } = useQuery({

    queryKey: ["services"],

    queryFn: async () => {

      const res = await fetch("/api/services");

      return res.json();

    },

  });

  return (

    <div className="container py-10">

      <h1 className="text-4xl font-bold mb-8">

        الخدمات القانونية

      </h1>

      <div className="grid md:grid-cols-3 gap-6">

        {data.map((service: any) => (

          <ServiceCard
            key={service.id}
            service={service}
          />

        ))}

      </div>

    </div>

  );

}
```

### أفضل طريقة للبناء هي على مراحل:

1. متجر الخدمات القانونية.
2. صفحة تفاصيل الخدمة.
3. السلة والدفع.
4. إنشاء الطلب والفاتورة تلقائياً.
5. بوابة العميل.
6. نظام حجز الاستشارات.
7. الدفع الإلكتروني (مدى وApple Pay وSTC Pay).
8. لوحة إدارة الخدمات داخل المكتب.

وهذا سيعطيك نظاماً قريباً من **LegalZoom + Clio Grow** داخل عدالة AI.
