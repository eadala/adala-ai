نعم، وهذه من أهم المزايا التي ستجعل **عدالة AI** منصة متعددة المكاتب (Multi-Tenant SaaS) حقيقية، بحيث يمتلك كل مكتب هويته الخاصة دون الحاجة لتعديل الكود.

## 1. جدول الهوية البصرية للمكاتب

### `shared/schema.ts`

```ts
export const officeSettings = pgTable("office_settings", {

  id: uuid("id").primaryKey().defaultRandom(),

  officeId: uuid("office_id")
    .references(() => offices.id)
    .notNull(),

  officeName: varchar("office_name", {
    length: 255
  }),

  logoUrl: text("logo_url"),

  stampUrl: text("stamp_url"),

  signatureUrl: text("signature_url"),

  primaryColor: varchar("primary_color", {
    length: 20
  }).default("#1e40af"),

  secondaryColor: varchar("secondary_color", {
    length: 20
  }).default("#f8fafc"),

  invoiceTemplate: varchar("invoice_template", {
    length: 50
  }).default("classic_legal"),

});
```

# 3. Hook لجلب إعدادات المكتب

### `hooks/useOfficeSettings.ts`

```ts
import { useQuery } from "@tanstack/react-query";

export function useOfficeSettings() {

  return useQuery({

    queryKey: ["office-settings"],

    queryFn: async () => {

      const response = await fetch(
        "/api/office-settings"
      );

      return response.json();

    },

  });

}
```

---

# 4. ThemeProvider

### `components/providers/OfficeThemeProvider.tsx`

```tsx
import { createContext } from "react";

export const OfficeThemeContext =
  createContext(null);

export function OfficeThemeProvider({
  settings,
  children,
}: any) {

  return (

    <div
      style={{
        ["--primary-color" as any]:
          settings.primaryColor,

        ["--secondary-color" as any]:
          settings.secondaryColor,
      }}
    >

      {children}

    </div>

  );
}
```
