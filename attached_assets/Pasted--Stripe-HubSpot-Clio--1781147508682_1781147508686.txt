# Monthly / Yearly Toggle

```tsx
<Card className="w-fit mx-auto p-1">

<div className="flex gap-2">

<Button
variant={
billingCycle === "monthly"
? "default"
: "ghost"
}
onClick={() =>
setBillingCycle("monthly")
}
>
شهري
</Button>

<Button
variant={
billingCycle === "yearly"
? "default"
: "ghost"
}
onClick={() =>
setBillingCycle("yearly")
}
>
سنوي
<span className="mr-2 text-green-500">

خصم 20%

</span>

</Button>

</div>

</Card>
```
