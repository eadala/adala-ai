🧠 ثانياً: الخدمات الأساسية (Microservices)

1) 🧠 AI Core Service (العقل القانوني)

الوظيفة:

* تحليل القضايا
* تشغيل الوكلاء
* إدارة الاستدلال القانوني

القيود:

* ❌ لا يكتب بيانات في الشعيب
* ✔ قراءة فقط

تقنية:

* Node.js / TypeScript
* LangGraph أو custom orchestration

⸻

2) ⚖️ Case Service (خدمة القضايا)

الوظيفة:

* إدارة metadata داخل المنصة فقط
* ربط القضايا بـ Shuaib Case ID

الجداول:

* internal_case_id
* shuaib_case_id (FK خارجي)
* office_id

⸻

3) 📄 Document Service

الوظيفة:

* توليد مستندات قانونية
* PDF generation
* Versioning

القيد:

كل مستند يجب أن يرتبط بـ shuaib_case_id

⸻

4) 🔗 Shuaib Adapter (أهم مكوّن)

الوظيفة:

واجهة اتصال آمنة مع نظام الشعيب

مبدأ أساسي:

Read-Only Adapter

export class ShuaibAdapter {
  async getCase(id: string) {
    return fetch(`${process.env.SHUAIB_URL}/cases/${id}`, {
      headers: {
        Authorization: `Bearer ${process.env.SHUAIB_TOKEN}`,
      },
    }).then(r => r.json());
  }
}

ممنوع:

* POST
* PUT
* DELETE

⸻

5) 🎨 UI Service (عرض البيانات فقط)

الوظيفة:

* بناء dashboards للقضايا
* عرض الجلسات
* عرض التحليلات

القاعدة:

UI لا ينشئ بيانات — فقط يعرض بيانات الشعيب

⸻

6) 🧠 AI Simulation Service

الوظيفة:

* توقع أسئلة القاضي
* محاكاة الخصم
* تحليل المخاطر

مدخلاته:

* Case snapshot (read-only)

⸻

7) 🔐 Auth & Governance Service

الوظيفة:

* صلاحيات المستخدمين
* تسجيل العمليات
* منع التلاعب

Roles:

* Judge Viewer
* Lawyer
* Office Admin
* AI System

⸻

🗄️ ثالثاً: قاعدة البيانات (PostgreSQL Multi-Tenant)

🧩 التصميم:

TABLE offices (
  id UUID PRIMARY KEY,
  name TEXT
);
TABLE cases (
  id UUID PRIMARY KEY,
  office_id UUID,
  shuaib_case_id TEXT UNIQUE,
  status TEXT
);
TABLE documents (
  id UUID PRIMARY KEY,
  case_id UUID,
  type TEXT,
  source TEXT DEFAULT 'shuaib'
);
TABLE ai_logs (
  id UUID PRIMARY KEY,
  case_id UUID,
  agent TEXT,
  output JSONB
);

🧠 رابعاً: تدفق النظام (Execution Flow)

User Request
   ↓
API Gateway
   ↓
Auth Check (Governance)
   ↓
Case Service (validate office + case)
   ↓
Shuaib Adapter (fetch truth data)
   ↓
AI Core Service
   ↓
Agent Execution
   ↓
Simulation / Document / Analysis
   ↓
Response Rendering

🛡️ خامساً: قواعد الأمان القانونية (Legal Governance)

📌 قاعدة 1

الشعيب هو المصدر الوحيد للحقيقة

📌 قاعدة 2

AI لا ينشئ قضايا ولا يعدّل بيانات

📌 قاعدة 3

كل عملية يتم تسجيلها في Audit Log

📌 قاعدة 4

أي بيانات بدون shuaib_case_id تعتبر باطلة
