🧠 2️⃣ Control Tower Engine (الدماغ)

📌 يحلل 3 أشياء:

1. الهوية

* unresolved tenants
* drift anomalies

2. الأمان

* RBAC failures spikes
* suspicious access

3. الأداء

* latency
* AI cost spikes

⸻

⚙️ الكود

export function analyzeEvents(events) {
  const alerts = [];
  const rbacFailures = events.filter(e => e.type === "RBAC_DENIED").length;
  if (rbacFailures > 20) {
    alerts.push({
      level: "warning",
      type: "RBAC_SPIKE"
    });
  }
  const tenantFailures = events.filter(e => e.status === "fail");
  if (tenantFailures.length > 5) {
    alerts.push({
      level: "critical",
      type: "TENANT_RESOLUTION_ANOMALY"
    });
  }
  return alerts;
}

🚨 3️⃣ Security Anomaly Detection

📌 يكشف:

* محاولة الوصول لمكتب آخر
* API misuse
* burst RBAC failures
* suspicious AI usage

⸻

⚙️ مثال

function detectSecurityThreat(event) {
  if (event.type === "TENANT_MISMATCH") {
    return {
      threat: "HIGH",
      reason: "Cross-tenant access attempt"
    };
  }
  if (event.type === "MULTI_RBAC_FAILURE") {
    return {
      threat: "MEDIUM",
      reason: "Permission probing detected"
    };
  }
  return null;
}

🧠 5️⃣ Health Scoring System

📌 كل 10 ثواني يتم حساب “صحة النظام”

export function systemHealthScore(metrics) {
  let score = 100;
  score -= metrics.rbacFailures * 0.5;
  score -= metrics.tenantErrors * 2;
  score -= metrics.latencySpike ? 10 : 0;
  return Math.max(score, 0);
}

⸻

📊 النتائج:

Score	الحالة
90–100	🟢 ممتاز
70–89	🟡 مستقر
50–69	🟠 خطر
<50	🔴 توقف مراقب
