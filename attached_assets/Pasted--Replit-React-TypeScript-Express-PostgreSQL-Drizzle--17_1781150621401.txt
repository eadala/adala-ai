## دالة التحقق من الخدمة

### shared/utils/hasFeature.ts

```ts
export function hasFeature(

officePlan: any,

feature: string

) {

return officePlan?.features?.[feature] === true;

}
```

---

## الاستخدام

### إظهار متجر الخدمات

```tsx
{
hasFeature(
office.plan,
"serviceStore"
)

&&

<MenuItem>

متجر الخدمات القانونية

</MenuItem>
}
```

---

```tsx

"website"

&&

}
```

## تنبيه عمولة الدفع الإلكتروني

داخل صفحة الدفع:

```tsx
<Alert>

<AlertDescription>

عند استخدام خدمات الدفع الإلكتروني عبر منصة عدالة AI يتم استقطاع عمولة تشغيل وإدارة بنسبة 10% من قيمة المدفوعات المحصلة، بالإضافة إلى رسوم مزود خدمة الدفع إن وجدت.

</AlertDescription>

</Alert>
```
