Critical RBAC Enforcement Fix

Priority: RELEASE BLOCKER

The RBAC audit identified critical privilege escalation vulnerabilities.

Tenant isolation is working correctly.

RBAC enforcement is not.

Do not add new features.

Focus only on RBAC remediation.

⸻

Phase 1 — Immediate Critical Fixes

Create centralized permission middleware

Implement:

requirePermission(permission: string)

The middleware must:

1. Resolve authenticated tenant user.
2. Load user role.
3. Load role permissions.
4. Verify requested permission.
5. Return 403 if denied.

No route may rely on frontend permission checks.

⸻
