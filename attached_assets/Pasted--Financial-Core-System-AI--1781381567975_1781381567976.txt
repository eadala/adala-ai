تمام. سنبني الآن النظام المالي الكامل (Financial Core System) لمنصة عدالة AI بحيث يتحول من “نظام دفع” إلى نظام محاسبي + Fintech Ledger + تسويات تلقائية.

هذا هو القلب الحقيقي لأي منصة سوق محامين + SaaS قانوني.

⸻

🧠 1. ما الذي نبنيه هنا؟

نظام مالي متكامل يشمل:

* Ledger محاسبي مزدوج (Double-entry accounting)
* Wallets لكل مستخدم
* Revenue split (عمولة المنصة + المحامي)
* تسويات دورية (Settlement Engine)
* تقارير مالية لحظية
* سجل تدقيق (Audit Log)
* دعم الاشتراكات + الفواتير + السوق

⚙️ 4. محرك الـ Ledger (الأهم)

📌 قاعدة ذهبية:

أي عملية مالية = قيد مزدوج (Debit / Credit)

⸻

🔧 تنفيذ Ledger Engine

export class LedgerService {
  async createEntry(data: {
    debitAccountId: string;
    creditAccountId: string;
    amount: number;
    transactionId: string;
    description: string;
  }) {
    // 1. إنشاء القيد
    const entry = await db.ledgerEntry.create({
      data: {
        ...data,
        currency: "SAR",
        createdAt: new Date(),
      },
    });
    // 2. تحديث الأرصدة
    await this.updateBalances(data);
    return entry;
  }
  async updateBalances(data: any) {
    await db.account.update({
      where: { id: data.debitAccountId },
      data: { balance: { decrement: data.amount } },
    });
    await db.account.update({
      where: { id: data.creditAccountId },
      data: { balance: { increment: data.amount } },
    });
  }
}

💳 5. معالجة الدفع (Webhook Processor)

app.post("/webhooks/payment", async (req, res) => {
  const event = req.body;
  if (event.type === "payment.succeeded") {
    const tx = await db.transaction.findById(event.data.reference);
    // 1. تحديث الحالة
    await db.transaction.update({
      where: { id: tx.id },
      data: { status: "paid" },
    });
    // 2. توزيع الأموال
    await FinancialEngine.processTransaction(tx);
  }
  res.sendStatus(200);
});

⚖️ 6. نظام توزيع الأرباح (Marketplace Split)

export class FinancialEngine {
  static async processTransaction(tx: any) {
    const platformFee = tx.amount * 0.10;
    const lawyerAmount = tx.amount - platformFee;
    const platformAccount = "platform_main";
    const lawyerAccount = tx.lawyerId;
    // قيد 1: العميل → المنصة
    await ledger.createEntry({
      debitAccountId: tx.userId,
      creditAccountId: platformAccount,
      amount: tx.amount,
      transactionId: tx.id,
      description: "Client payment",
    });
    // قيد 2: المنصة → المحامي
    await ledger.createEntry({
      debitAccountId: platformAccount,
      creditAccountId: lawyerAccount,
      amount: lawyerAmount,
      transactionId: tx.id,
      description: "Lawyer payout",
    });
    // تسجيل الأرباح
    await db.payout.create({
      lawyerId: tx.lawyerId,
      amount: lawyerAmount,
      status: "pending",
    });
  }
}

💰 7. محرك التسويات (Settlement Engine)

export class SettlementEngine {
  static async runDailySettlement() {
    const payouts = await db.payout.findMany({
      where: { status: "pending" },
    });
    for (const payout of payouts) {
      await this.processPayout(payout);
    }
  }
  static async processPayout(payout: any) {
    // إرسال إلى البنك أو مزود الدفع
    const result = await PaymentProvider.payout({
      amount: payout.amount,
      recipient: payout.lawyerId,
    });
    if (result.success) {
      await db.payout.update({
        where: { id: payout.id },
        data: { status: "sent" },
      });
    }
  }
}
