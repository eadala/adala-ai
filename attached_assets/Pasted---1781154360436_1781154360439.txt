# Adala Studio

### Low-Code / No-Code Builder

بحيث تستطيع إنشاء الأقسام والحقول والنماذج وسير العمل من داخل المنصة بدون تعديل الكود.

---

# هيكل النظام

```text
Adala Studio
│
├── Database Studio
├── Page Builder
├── Form Builder
├── Workflow Builder
├── AI Developer
├── Theme Builder
├── Plugin Store
├── API Center
└── DevOps Center
```

# 1- الجداول الديناميكية

## shared/schema/customTables.ts

```ts
export const customTables = pgTable(
"custom_tables",
{

id: uuid("id")
.defaultRandom()
.primaryKey(),

officeId: uuid("office_id"),

tableName: varchar(
"table_name",
{
length:255
}
).notNull(),

displayName: varchar(
"display_name",
{
length:255
}
),

icon: varchar(
"icon",
{
length:100
}
),

createdAt: timestamp(
"created_at"
).defaultNow()

}
);
```

## أكثر ميزة أوصي بها مستقبلاً

إضافة:

```text
Adala AI Developer
```

بحيث تكتب:

> أضف قسم المصاريف

أو:

> أنشئ صفحة للعقارات

أو:

> أضف تقرير الأرباح والخسائر

فيقوم النظام تلقائياً بإنشاء:

* Drizzle Schema
* API Routes
* Services
* React Pages
* الصلاحيات
* عناصر القائمة

وهذه من أقوى المزايا التي يمكن أن تجعل عدالة AI منصة قابلة للتطوير الذاتي وتقلل اعتمادك على المبرمجين بشكل كبير.
