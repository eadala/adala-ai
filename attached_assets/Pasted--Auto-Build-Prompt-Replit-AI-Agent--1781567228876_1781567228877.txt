🏛️ Architecture Rules (مهم جداً)

1) النظام Modular Monolith (ليس Microservices)

كل الموديولات داخل نفس المشروع:

* cases
* clients
* finance
* documents
* ai
* hr
* notifications

⸻

2) يجب تطبيق Event-Driven Design داخلي

أنشئ Event Bus بسيط:

emit(eventName, payload)
on(eventName, handler)

أمثلة Events:

* CaseCreated
* ContractSigned
* InvoiceIssued
* TaskCreated

⸻

3) AI Layer MUST be isolated

كل الذكاء الاصطناعي يمر عبر:

POST /api/ai/query

ولا يُستدعى مباشرة من UI.

يجب بناء:

* AI Router
* Agent Manager
* Response Cache

⸻

4) Performance First (مهم جداً)

يجب تطبيق:

* Lazy loading لكل الصفحات
* Pagination لكل lists
* No full data fetch on dashboard
* Caching layer (in-memory or Redis-ready)

🧱 Database Design Rules

Use relational DB (PostgreSQL preferred)

Must include:

* cases
* clients
* contracts
* invoices
* users
* events_log

All tables must include:

* created_at
* updated_at
* tenant_id (multi-tenant ready)

🔁 Event System Implementation

Every mutation must emit event:

Example:

await caseService.create(data)
emit("CaseCreated", case)

🧠 AI RULES

* AI responses must be cached for 10 minutes minimum
