بما أنك تعمل على **Replit (React + TypeScript + Express + PostgreSQL + Drizzle)**، فمن الأفضل بناء النظام بطريقة احترافية وقابلة للتوسع، بحيث تكون **الرواتب مرتبطة بالمصاريف والتقارير المالية تلقائياً**.

# 2) ربط الرواتب بالمصاريف تلقائياً

## `server/services/payrollService.ts`

```ts
import { db } from "../db";
import {
  payrolls,
  expenses
} from "@shared/schema";

export async function createPayroll(
  data: any
) {

  const salary = await db
    .insert(payrolls)
    .values(data)
    .returning();

  await db
    .insert(expenses)
    .values({

      officeId: data.officeId,

      title:
        `راتب الموظف ${data.employeeId}`,

      category: "الرواتب",

      amount: data.netSalary,

      paymentMethod: "bank",

      notes:
        `راتب شهر ${data.salaryMonth}`

    });

  return salary;
}
```

# 3) حساب الأرباح والخسائر

## `server/services/financeService.ts`

```ts
export async function getProfitLoss(
  officeId: string
) {

  const revenueTotal = 500000;

  const expenseTotal = 180000;

  return {

    revenue: revenueTotal,

    expenses: expenseTotal,

    netProfit:
      revenueTotal -
      expenseTotal

  };

}
```

# 4) API

## `server/routes/finance.ts`

```ts
router.get(
"/profit-loss",

req,
res
) => {

await getProfitLoss(

res.json(data);

});
```
