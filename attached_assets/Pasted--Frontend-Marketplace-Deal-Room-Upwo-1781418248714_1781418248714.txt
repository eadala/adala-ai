تمام — الآن سننتقل للمرحلة التي تجعل النظام “حي”: Frontend Marketplace + Deal Room (مثل خمسات + Upwork) لكن مخصص لعدالة AI ومربوط مباشرة بالـ Backend الذي بنيناه.

سأعطيك تصميم + هيكل + كود أساس جاهز لبداية تنفيذ فعلية.

⸻

🎨 ⚖️ عدالة AI — Frontend (Next.js Production UI)

🧠 الهدف

بناء 3 واجهات رئيسية:

1. Public Landing (تعريفي)
2. Marketplace (مثل خمسات)
3. Deal Room (التفاوض + العقد)
4. Client Dashboard (القضايا)

🛒 3) Marketplace Page (مثل خمسات)

🔥 أهم صفحة في النظام

export default async function MarketplacePage() {
  const services = await fetch("API/services").then(r => r.json())
  return (
    <div className="p-6">
      <h1 className="text-2xl font-bold mb-6">
        الخدمات القانونية
      </h1>
      <div className="grid grid-cols-3 gap-4">
        {services.map((s: any) => (
          <a key={s.id} href={`/marketplace/service/${s.id}`}

            <p className="text-sm text-gray-500">{s.description}</p>
            <div className="mt-2 font-bold">
              {s.price} SAR
            </div>
          </a>

      </div>

  )
}

📄 4) Service Page (قرار الشراء)

export default async function ServicePage({ params }: any) {
  const service = await fetch(`/api/services/${params.id}`).then(r => r.json())
  return (
    <div className="p-8">
      <h1 className="text-3xl font-bold">{service.title}</h1>
      <p className="mt-4">{service.description}</p>
      <div className="mt-6 text-xl font-bold">
        {service.price} SAR
      </div>
      <a
        href={`/checkout?service=${service.id}`}
        className="mt-6 inline-block btn-primary"
      >
        اطلب الخدمة الآن
      </a>
      <a
        href={`/deals/${service.id}`}
        className="ml-4 btn-secondary"
      >
        تفاوض على السعر
      </a>
    </div>
  )
}
