واضح أنك تريد الانتقال من “نصائح تصميم” إلى نظام جاهز للتطبيق مباشرة (UI Kit عملي). سأبنيه لك بشكل منظم بحيث تقدر تنقله إلى أي مشروع (Replit / React / Next / Vue / حتى HTML فقط).

⸻

🧩 UI KIT كامل لمنصة “عدالة”

🧭 1) الأساس: Design Tokens (العمود الفقري)

🎨 الألوان

:root {
  --primary: #1E3A8A;      /* أزرق قانوني ثقة */
  --secondary: #64748B;    /* رمادي احترافي */
  --bg: #F8FAFC;           /* خلفية عامة */
  --card: #FFFFFF;         /* الكروت */
  --border: #E2E8F0;
  --success: #16A34A;
  --warning: #F59E0B;
  --danger: #DC2626;
}

⸻

🔤 الخطوط

body {

}

🧱 2) Card System (جاهز للاستخدام)

🔹 Base Card (أساس كل الكروت)

.card {
  background: var(--card);
  border: 1px solid var(--border);
  border-radius: var(--radius-md);
  padding: 16px;
  transition: all 0.2s ease;
}
.card:hover {
  transform: translateY(-2px);
  box-shadow: 0 6px 18px rgba(0,0,0,0.06);
}

⸻

⚠️ أهم إصلاح لمشكلتك السابقة

المشكلة:

* كرت كبير + خط صغير

الحل داخل النظام هذا:

✔ الخط داخل الكرت = 15–18px
✔ padding = 16px
✔ لا height ثابت
✔ كل كرت فيه hierarchy واضحة

⸻

🚀 النتيجة بعد تطبيق هذا UI KIT

ستحصل على:

* واجهة “قانونية احترافية” مثل SaaS عالمي
* توازن بصري مضبوط
* كروت غير “فارغة بصريًا”
* خط واضح ومقروء
* تجربة تشبه أنظمة مثل Clio / Notion / Stripe dashboards
