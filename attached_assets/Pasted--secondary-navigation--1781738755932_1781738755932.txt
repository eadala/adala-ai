هذا طبيعي لأن القائمة الجانبية غالبًا تكون مصممة كـ “secondary navigation”، لكن في حالتك (منصة قانونية مثل عدالة) لازم تتحول إلى واجهة مؤسسية واضحة وقوية بصريًا وليس نص صغير ضعيف.

سأعطيك تعديل مباشر يرفعها مستوى كامل.

⸻

🧭 تحسين القائمة الجانبية (Sidebar) — مستوى احترافي فاخر

1) المشكلة الحالية

غالبًا عندك:

* خط صغير (12–13px)
* وزن خفيف (400)
* تباعد ضعيف
* لا يوجد إبراز للعنصر النشط
* إحساس “تطبيق عادي” وليس نظام مؤسسي

⸻

🚀 الحل الفوري (Sidebar Typography Upgrade)

✨ الحجم والوزن الجديد

.sidebar-item {
  font-size: 14.5px;
  font-weight: 600;
  line-height: 1.6;
  letter-spacing: -0.01em;
}

⸻

✨ تحسين العناصر (Spacing + Padding)

.sidebar-item {
  padding: 10px 14px;
  border-radius: 10px;
  transition: all 0.2s ease;
}

⸻

✨ العنصر النشط (Active State) — مهم جدًا للفخامة

.sidebar-item.active {
  background: rgba(17, 24, 39, 0.06);
  color: #111827;
  font-weight: 700;
  border-right: 3px solid #111827;
}

⸻

✨ تحسين الأيقونات مع النص

.sidebar-item svg {
  width: 18px;
  height: 18px;
  margin-left: 10px;
  opacity: 0.85;
}

⸻

✨ Hover احترافي (يعطي إحساس نظام كبير)

.sidebar-item:hover {
  background: rgba(17, 24, 39, 0.04);
  transform: translateX(-2px);
}

⸻

🧠 أهم نقطة (فرق الأنظمة الاحترافية)

❌ تصميم ضعيف:

* خط 12px
* وزن 400
* بدون active واضح

✅ تصميم مؤسسي:

* 14–15px
* وزن 600–700
* active indicator واضح
* spacing مريح
* حركة hover خفيفة

⸻

🏛️ رفع مستوى “عدالة” إلى UI قانوني فعلي

لو تبغى نفس مستوى الأنظمة القضائية:

نضيف:

* Sidebar ثابت مع shadow خفيف
* تقسيم أقسام (إدارة / محاسبة / قضايا)
* separators بين المجموعات
* badges (مثل: جديد / تنبيه)

.sidebar-group-title {
  font-size: 11px;
  font-weight: 700;
  color: #9ca3af;
  margin: 16px 0 6px;
  letter-spacing: 0.05em;
}

⸻

⚖️ النتيجة بعد التعديل

* القائمة تصبح واضحة بصريًا
* إحساس “نظام قانوني مؤسسي”
* قراءة أسرع
* تمييز أقوى بين الأقسام
* تجربة أقرب لأنظمة ERP القضائية

⸻

🚀 إذا تريد الخطوة التالية

أقدر أرفع لك مستوى الـ Sidebar أكثر إلى:

* ￼ تصميم “Admin Panel حكومي”
* أو ￼ نسخة “Lux Legal UI” فخمة جدًا
* أو ￼ إعادة تصميم كاملة للـ Dashboard navigation

فقط قل:
“ارفع القائمة الجانبية مستوى حكومي”
