## 2. القائمة الرئيسية

### `sidebarItems.ts`

```ts
import { financialAccountingItems } from "./financialItems";

export const sidebarItems = [

  {
    title: "الإدارة المالية",
    icon: "DollarSign",
    children: [

      {
        title: "الفواتير",
        href: "/invoices"
      },

      {
        title: "الاشتراكات والفوترة",
        href: "/subscriptions"
      },

      {
        title: "المركز المالي",
        href: "/finance-dashboard"
      },

      {
        title: "المحاسبة المالية",
        children: financialAccountingItems
      }

    ]
  }

];
```

# إذا كانت العناصر تظهر مرتين بسبب خطأ في الـ JSX

ابحث عن:

```tsx
<SidebarGroup title="المحاسبة المالية">
    {financialAccountingItems.map(renderMenu)}
</SidebarGroup>

<SidebarGroup title="المحاسبة المالية">
    {financialAccountingItems.map(renderMenu)}
</SidebarGroup>
```

واحذف المجموعة الثانية.

# الهيكل الاحترافي الذي أوصي به لعدالة AI

```text
الإدارة المالية
│
├── الفواتير
├── الاشتراكات والفوترة
├── المركز المالي
│
└── المحاسبة المالية
     ├── الإيرادات
     ├── المصاريف
     ├── التدفقات النقدية
     ├── الحسابات البنكية
     ├── العهد والسلف
     ├── القيود اليومية
     ├── سندات القبض
     ├── سندات الصرف
     ├── الرواتب
     ├── الأرباح والخسائر
     └── التقارير المالية
```

وبذلك ستظهر قائمة **المحاسبة المالية مرة واحدة فقط** بدون أي تكرار.
