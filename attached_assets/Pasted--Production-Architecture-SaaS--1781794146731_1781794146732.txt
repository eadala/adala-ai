📊 7) Key Design Rules (مهم جداً)

🔴 قاعدة 1

لا يوجد أي query بدون office_id

🔴 قاعدة 2

لا يوجد AI بدون tenant filter

🔴 قاعدة 3

لا يوجد file access بدون signed URL

🔴 قاعدة 4

كل API محمي بـ middleware

🔴 قاعدة 5

كل حدث مسجل في audit logs
